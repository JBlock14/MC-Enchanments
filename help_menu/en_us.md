This platform gathers all the enchantments from the game MINECRAFT in one place.  
The information is presented in a clear and straightforward way:

- **Enchantment name:** Official identification.  
- **Basic description:** Main effect within the game.  
- **Item compatibility:** Tools, weapons, or armor pieces that can receive it.  

The goal is to provide a simple and accessible reference, with fast navigation and optimized design, serving as a micro Wiki-Web for players seeking simplicity.

---

### Search Bar
You can use the integrated search bar to navigate more comfortably and find the enchantment you want more easily.

On computers, or any device with a technical keyboard, you can quickly select the search bar by pressing the TAB key once, unless it is already selected.

This is not the only utility of the search bar, as it also includes shortcuts and commands that improve web exploration.

#### Commands:
Interaction commands are frequently used to view extra information or have more control. They are always written in lowercase.

**`/help`**: Opens the current menu where you will find complete information about the website: utilities, purpose, syntax, and more.

#### Tags:
Tags, or search filters, are filters used to search for enchantments of a specific tool, weapon, or armor piece and limit the list to specific options.  
They work like filters, but instead of filtering by a group, they filter by specific items.  
You can activate several at once.

The function is simple: you just need to open the tag with **:** (colon), type the name of the item you want to filter (you can usually see it by hovering the mouse over the icon or tapping it on touch screens), and close again with **:**.  
Try not to use unnecessary spaces.

### Filter Button
Filtering works as an option to organize the list by a specific group of items.  
When activated, it will display a menu with the following options:

- **Armor filter:** Shows all possible enchantments for armor pieces.  
- **Weapons filter:** Shows all possible enchantments for weapons.  
- **Ranged filter:** Shows all possible enchantments for ranged weapons.  
- **Tools filter:** Shows all possible enchantments for tools.
